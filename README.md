# GymH-iOS
GymH iOS App

- written entirely in XCode
- Swift 4
- Storyboards

## Features

### Vertretungsplan
- displays https://vpgh.lima-city.de/
- optionally switches to https://vpgh.lima-city.de/week.php
- optionally displayed with filters + theming

### Termine
- displays https://vpgh.lima-city.de/termine/

### Settings
- theming
- vp filter
- week/day switch
